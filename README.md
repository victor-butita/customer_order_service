# Customer Order Service

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/customer-order-service.git


Install dependencies:

pip install -r requirements.txt
Set up Africa's Talking API credentials in config.py.

Run the app:

flask run
API Endpoints
POST /customers: Add a new customer.
POST /orders: Add a new order and send an SMS alert to the customer.
Testing
Run the unit tests:

pytest --cov=app
CI/CD
CI/CD is set up via GitHub Actions in .github/workflows/ci_cd.yml.

vbnet
